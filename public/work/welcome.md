# Welcome

This is an example work entry.

## How to add your own work

1. Drop a Markdown file into `public/work/`.
2. Optionally add frontmatter:

```yaml
---
title: My Project
date: 2026-02-13
description: A short summary.
---
```

3. Run `npm run dev` and it will show up on the home page.
